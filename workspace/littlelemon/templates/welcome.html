Below are the API endpoints you can test:

  • Menu API - View and create menu items.
  • Booking API - View and create reservations.

Note: To view menu items and reservations, create them first through POST requests.

Booking API

Use this JSON to create a new booking:

{
    "name": "John Doe",
    "no_of_guests": 4,
    "booking_date": "2025-03-22T11:00:00Z"
}
